# EasyKash

## before run backend 
 you should creat Mysql database called  `easyKash `
 and create tables `offers  ,sellers`
 you can import data from folder `dbResources` to  `easyKash ` database
 
## Run Angular server
Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

 

## Run Nodejs Backend
Run `nodemon server.ts` to start backend for the project.


## Running unit test and integration test
  please run `run npm test `
